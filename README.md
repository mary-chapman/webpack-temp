# webpack-template

## To start

* clone the repo: `$ git clone https://github.com/mary-chapman/webpack-template.git`
* go into the directory: `$ cd webpack-template`
* install dependencies: `$ npm install`
* run: `$ npm run start`
* application will run on http://localhost:8080

## Contains: 

* webpack
* webpack-dev-server
* hot reloader
* react
* babel
* css
* sass
* bootstrap
* image files
* font-awesome
